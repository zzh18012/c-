# 困难/超困难模式设计方案

## 日期
2026/05/30

## 概述
为游戏增加困难和超困难模式，分别对应2只和3只BOSS同时在场。每只BOSS拥有独特外观、背景特效、出场动画和弹幕风格。

---

## BOSS设计

### Boss A — 烈焰恶魔 (Inferno Demon)

| 属性 | 描述 |
|------|------|
| **外观** | 红/橙色恶魔，燃烧的角 + 鳞片，发光眼睛 |
| **光环特效** | 火焰环绕，燃烧的余烬粒子向外飘散 |
| **出场动画** | 从屏幕底部向上升起，火焰拖尾 + 燃烧粒子特效 |
| **弹幕风格** | 火球散射、追踪火球、爆发燃烧弹幕 |

### Boss B — 虚空幽灵 (Void Specter)

| 属性 | 描述 |
|------|------|
| **外观** | 紫/蓝色幽灵体，半透明，不断变形蠕动 |
| **光环特效** | 虚空能量场，漂浮的星尘粒子围绕 |
| **出场动画** | 空间撕裂效果，屏幕出现裂缝，Boss从裂缝中缓缓涌现 |
| **弹幕风格** | 暗影弹幕、虚空球体、全屏暗影脉冲 |

### Boss C — 雷霆泰坦 (Thunder Titan)

| 属性 | 描述 |
|------|------|
| **外观** | 黄/金色机械泰坦，电弧在表面不断跳跃 |
| **光环特效** | 闪电链环绕，电流粒子在周围窜动 |
| **出场动画** | 脉冲爆发冲击波，泰坦从电光中成形扩散 |
| **弹幕风格** | 闪电链弹幕、电弧散射、落雷攻击 |

---

## 难度模式

### 困难模式（2只BOSS）

- **BOSS组合**：烈焰恶魔(A) + 虚空幽灵(B)
- **入场方式**：游戏开始时两只BOSS同时入场
- **血条显示**：2根血条
- **胜利条件**：击败所有2只BOSS

### 超困难模式（3只BOSS）

| 阶段 | 触发条件 | BOSS状态 |
|------|----------|----------|
| 第一阶段 | 0% | Boss A 满血入场 |
| 第二阶段 | Boss A 血量 ≤ 50% | Boss B + Boss C 同时入场（各满血） |
| 第三阶段 | — | 3只BOSS同时战斗直到全部击败 |

- **血条显示**：3根血条
- **胜利条件**：击败所有3只BOSS

---

## 数值设计

| 参数 | 数值 |
|------|------|
| BOSS HP | 各BOSS独立计算，HP相同（不随难度调整） |
| 玩家血量 | 不调整 |
| BOSS位置 | 随机分布 |

---

## 系统设计

### BOSS管理器
- 支持同时管理多个BOSS实例
- 根据难度模式初始化不同数量的BOSS
- 处理BOSS入场逻辑（同时入场 / 分批入场）

### 血条UI
- 动态显示当前存活的BOSS血条
- 困难模式：最多2根
- 超困难模式：最多3根
- 血条顺序对应BOSS出场顺序

### BOSS识别系统
- 每只BOSS拥有独特的视觉标识
- 背景光环帮助玩家快速识别正在攻击的BOSS
- 即使弹幕交织也能分辨目标
